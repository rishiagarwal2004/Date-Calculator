# 📅 Date Calculator

A simple and responsive **Date Calculator** web application that calculates the difference between two dates in **Years, Months, and Days**.

This project is built using **HTML, CSS, and JavaScript** and provides a clean user interface with responsive design.

---

## 🚀 Features

* 📆 Select **Start Date** and **End Date**
* 🔢 Calculates difference in **Years, Months, and Days**
* ⚡ Instant calculation with JavaScript
* 🎨 Clean and responsive UI
* ⌨️ Keyboard support (Press **Enter** to move between fields)
* 📱 Works on **desktop and mobile screens**

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and responsive layout
* **JavaScript (Vanilla JS)** – Date calculation logic and DOM manipulation

## ⚙️ How It Works

1. User selects a **Start Date** and **End Date**.
2. When the **Calculate Date** button is clicked:

   * JavaScript reads both input values.
3. The program calculates the difference between the two dates:

   * Years
   * Months
   * Days
4. The result is displayed on the screen.

Example Output:

```
3 Years 2 Months 10 Days
```
